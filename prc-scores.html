PRC Dist Scoring

This is a sortable & searchable table of the CPAN distributions that are currently candidates (ie not yet assigned) for the CPAN pull request challenge. There are more dists with a github repo given in the metadata, but this only contains dists where the repo actually exists. The scoring scheme is described in a blog post.
